🏀 Basketball Scoreboard
A simple basketball scoreboard built using HTML, CSS, and JavaScript.
This project allows users to keep track of scores for two teams (Home and Guest). It can be used for personal games, practice sessions, or small team matches.

🚀 Features
Add +1, +2, +3 points for both teams
Reset the scoreboard instantly
Digital-style scoreboard display using Google Fonts
Clean and minimal UI
Pure JavaScript (no frameworks)

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Google Fonts (Orbitron & Digital-7)

📸 Preview
The scoreboard includes:
Two team sections (Home & Guest)
Digital-style score display
Score increment buttons
Reset button

📂 How It Works
Each team has its own score variable (countL and countR).
Clicking a button increases the score.
The display updates using textContent.
The reset button sets both scores back to zero.
